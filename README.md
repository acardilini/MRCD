# MRCD Interactive Web Experience

A dynamic, highly personalized digital platform designed to automate the psychological profiling of Meat-Related Cognitive Dissonance (MRCD).

## Features
- **Interactive Scrollytelling:** Responsive design prioritizing a smooth user experience.
- **Dynamic Node Schematic:** Visualizes cognitive defenses in real-time.
- **Configurable Logic:** Completely driven by a static JSON framework, allowing researchers to swap triggers easily.
- **PDF Report Generation:** Provides users with an in-depth analysis of their MRCD persona.
